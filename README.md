# Amazon Home Page Clone

A responsive clone of the Amazon homepage built using **HTML** and **CSS**. This project focuses on recreating the layout and styling of the original website to improve frontend development skills.

##  Features

- Amazon-inspired homepage layout
- Responsive navigation bar
- Search bar with category selector
- Hero section
- Product category cards
- Footer similar to Amazon's homepage
- Responsive design using Flexbox

##  Technologies Used

- HTML
- CSS


##  How to Run

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional setup or dependencies are required.

##  What I Learned

This project helped me practice:

- Structuring webpages using HTML
- Styling layouts with CSS
- Using Flexbox for responsive layouts
- Positioning and alignment of elements
- Creating reusable UI sections
- Improving webpage design and organization
